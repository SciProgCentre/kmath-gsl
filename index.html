kmath-gsl

KMath extension adding GNU Scientific Library based linear algebra implementation.

  • src/nativeMain/kotlin/GslLinearSpace.kt : LinearSpace implementations for Double, Float, and Complex matrices and vectors implemented with GSL

Requirements:

kmath-gsl needs a BLAS implementation installed. To achieve better performance, MKL or ATLAS can be used.
